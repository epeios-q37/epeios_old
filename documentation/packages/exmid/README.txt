'exmid' README by Claude SIMON.
------------------------------------------------------
'exmid' is a MIDI file to XML converter from the Epeios project (http://zeusw.org/epeios/).

'exmid' is published under GNU General Public License (see LICENSE file).

The windows binary is provided with this package ('exmid.exe').

Under GNU/Linux, cygwin (and theoritically under Mac), unpack the archive, then launch 'make'.

Launch 'exmid --help' to obtain a help page.

See 'http://zeusw.org/intl/exmid/' for more details.
