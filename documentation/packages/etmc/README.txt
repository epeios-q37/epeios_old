Libraries dedicated to timecode handling.
Backend-related libraries (those with the tailing underscore) need the libraries from the 'ebkd' project (see 'http://zeusw.org/epeios/ebkd').

To learn more about this libraries, go to 'http://zeusw.org/epeios/'.
