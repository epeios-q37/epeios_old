'etmccalc' README by Claude SIMON.
------------------------------------------------------
'etmccalc' is a timecode calculator from the Epeios project (http://zeusw.org/epeios/).

'etmccalc' is published under GNU General Public Licesnse (see LICENSE file).

The windows binary is provided with this package ('etmccalc.exe').

Under GNU/Linux, cygwin (and theoritically under Mac), unpack the archive, then launch 'make'.

Launch 'etmccalc --help' to obtain a help page.

See 'http://zeusw.org/intl/etmccalc/' for more details.
