'expp' est un processeur XML.

L'exécutable windows est fourni dans l'archive ('expp.exe').

Sous GNU/Linux (ou cygwin), désarchiver l'archive, puis lancer la commande 'make'.

Voir 'http://zeusw.org/epeios/software/expp/'.
