$Id$

The Epeios libraries are not dedicated to a special domain. Memory management, error handling and frontend/backend interaction are some of the features available. You can also handle sockets, multitasking, semaphores, pipes, etc., and various data structures like lists, stacks, queues, indexes, etc.

To learn more about this libraries, go to 'http://epeios.org/'.
